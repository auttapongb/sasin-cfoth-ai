🔷 Key Frameworks

Core mental models and methodologies for this course:

Disruptive Innovation Theory (Christensen)

Established companies fail because what made them successful blinds them to disruption. Low-end disruption (target overserved customers) and New-market disruption (target non-consumers). The dilemma: serving current customers well can be fatal.

Platform vs. Pipeline Business Models

Pipelines create value through linear chains (design → make → sell). Platforms orchestrate value exchange between producers and consumers. The shift: value moves from controlling resources to facilitating interactions. Key challenge: solving the chicken-and-egg problem.

Network Effects

Same-side (more users = more value for same user type) and Cross-side (more drivers = better experience for riders). Dynamics: multi-homing weakens network effects, winner-take-all markets, the danger of congestion. Metcalfe's Law: network value ∝ n².

Business Model Canvas (Osterwalder)

9 building blocks: Customer Segments, Value Propositions, Channels, Customer Relationships, Revenue Streams, Key Resources, Key Activities, Key Partnerships, Cost Structure. Essential visual language for digital business model design.

Digital Transformation Maturity Model

Three phases: Digitization (analog → digital records), Digitalization (processes become digital), Digital Transformation (business model fundamentally changes). Spans customer experience, operations, AND strategy — requires all three.

🏢 Case Studies

Netflix — Textbook Disruptive Innovation
DVD-by-mail (low-end disruption of Blockbuster) → streaming → original content. Now the incumbent facing disruption from Disney+, HBO Max. The cycle continues.
Airbnb — New-Market Disruption
Created peer-to-peer short-term rental category. Unlocked idle capacity. Trust mechanisms (reviews, verified ID) solved the stranger-danger problem. Asset-light: owns no hotels.
Uber — The "Dangerous" Economics
Removed taxi friction (hailing, payment, uncertainty). Surge pricing coordinates supply and demand algorithmically. 14-year path to sustained profitability — the danger: high fixed costs, high CAC, low switching costs for drivers.
Shopify — Democratizing E-Commerce
Platform-as-a-Service for merchants. Evolved from shopping cart to complete commerce OS (payments, POS, logistics, lending). API ecosystem and App Store create powerful lock-in.
